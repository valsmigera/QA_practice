1. Отобразить подключённый девайс в консоли.
/adb devices
2. Вывести адрес приложения todolist в системе Android
/adb shell "pm list packages -f | grep 'todolist'"	#GREP PowerShell
/adb shell 'pm list packages -f' | Select-String todolist	#Select-string PShell
/adb shell 'pm list packages -f' | grep todolist	#linux
3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB
adb install D:\todolist.apk
4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде.
adb shell "screencap /storage/emulated/0/DCIM/1.png"; adb pull /storage/emulated/0/DCIM/1.png 
C:\Users\Val\1.png	#PShall
adb shell screencap /storage/emulated/0/DCIM/1.png && adb pull /storage/emulated/0/DCIM/1.png 
C:/Users/Val/1.png	#Linux
5. Вывести в консоль логи приложения todolist
/adb logcat | Select-String 'todolist'		#PowerShell
adb logcat | grep -rnw 'com.android.todolist'	#linux
6. Скопировать логи приложения todolist на компьютер.
/adb logcat | Select-String 'todolist' > log_todolist.log
7. Удалить приложение todolist с телефона через ADB
/adb uninstall com.android.todolist
